# 端口约定

后端服务端口从 `5301` 开始递增。

| 服务 | 目录 | 端口 |
| --- | --- | --- |
| 身份与权限服务 | `services/identity-service` | `5301` |
| 训练营业务服务 | `services/camp-service` | `5302` |
| 通知提醒服务 | `services/notification-service` | `5303` |

## 预留规则

- 新增后端服务继续使用 `5304`、`5305` 等端口。
- 前端开发端口暂不固定，等技术栈确定后再配置。
- 端口变更必须同步更新本文件和对应服务 `README.md`。
